Carousel Bootstrap using Nextgen Gallery.

== Description ==

English
--------

This plugin can display in a carousel Bootstrap all photos from any gallery of plugin Nextgen and all through a widget. For everything goes well I recommend use good quality photos and also having (all the photos) the same width and the same height.

Spanish
--------
Este plugin puede mostrar en un carrusel de Bootstrap todas las fotos de cualquier galeria de Nextgen Gallery y todo mediante un widget. Para que todo salga bien recomiendo usar fotos de buena calidad y ademas que tengan (todas) el mismo ancho y el mismo alto.


== Installation ==

1. Upload `widget-bootstrap-carousel-using-netxgen-gallery.zip` to the `/wp-content/plugins/` directory
2. Unzip the plugin
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

English
--------

= What can I customize? =

- Bootstrap version
- Include or not include bootstrap libraries in the header
- Transition time of each slide
- Choose the Nextgen Gallery to display
- Carousel ID
- Caption the photo


Spanish
--------

= Que puedo personalizar? =

- Version de Bootstrap
- Incluir o no incluir las librerias de bootstrap en la cabecera
- Tiempo de transicion de cada slide
- Elegir que galeria de Nextgen presentar
- ID del Carousel
- Subtitulo de cada foto
