Agrega una galeria de imagenes a tu sitio web y que sus visitantes las vean con solo pulsar un boton.

== Description ==

Este plugin te permitira tener una lista de tus imagenes e irlas visualizando una a una. Idóneo para artistas que quieran mostrar sus trabajos, albunes de fotos, casas de alquiler u hoteles, etc.

Si quieres ver un ejemplo de su funcionamiento, <a href="http://juanmacivico.com/galeria-de-imagenes/" target="_blank">pincha aqui</a>.

= Caracteristicas =

* Permite ver las imagenes, seleccionandolas una a una o pasandolas mediante los controles de avanzar y retroceder.
* Permite almacenar imagenes del tipo GIF, JPG o PNG, con un tamaño no superior a 5 MB.
* Permite eliminar las imagenes que ya no quieres mostrar.

== Installation ==

Para instalar el plugin, sigue estos pasos:

1. Descarga el plugin y descomprime su contenido.
2. Sube el contenido descomprimido (carpeta "galeria_imagenes") al directorio de plugins (/wp-content/plugins/).
3. Activa el plugin a traves del menu de plugins en el panel de control de WordPress.
4. Ve al nuevo apartado en el panel de control, llamado "GaleriaImagenes" y agrega o elimina las imagenes que desees.
5. Crea una nueva página o un post e inserta el codigo [insert_img] donde quieres que se muestre tu galeria de imagenes.
